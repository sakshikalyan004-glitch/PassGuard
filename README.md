# PassGuard

## Password Strength Checker and Analyzer

PassGuard is a Python-based password strength checker that analyzes passwords and classifies them as Weak, Medium, or Strong.

## Features
- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Detects common weak passwords
- Gives suggestions to improve password strength

## Technologies Used
- Python 3
- re
- getpass

## How to Run

```bash
python password_checker.py
```

## Sample Results
- 12345 → WEAK
- Pass123 → MEDIUM
- P@ssw0rd!2026 → STRONG
